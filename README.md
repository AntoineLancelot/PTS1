# PTS1 - Site web de sensibilisation

<br/>
<br/>


Dans le cadre d'un projet universitaire, nous avons dû réaliser un site internet de sensibilisation qui devait porté sur deux aspects : l'informatique et le handicap. Notre groupe avions choisi : la réalité virtuelle et augmentée au service de la cécité.
